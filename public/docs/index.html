Introduction

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

POST Add Picture.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/image" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "sort=999" \
    --form "path=@C:\Users\oguz\AppData\Local\Temp\php9EBE.tmp" 
const url = new URL(
    "http://127.0.0.1:8000/api/image"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('sort', '999');
body.append('path', document.querySelector('input[name="path"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

Example response (200):


{
    "path": "images/oguz.png",
    "sort": 999,
    "status": 1
}

DELETE Remove Picture.

Example request:
curl --request DELETE \
    "http://127.0.0.1:8000/api/image/31" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"id\": 18
}"

const url = new URL(
    "http://127.0.0.1:8000/api/image/31"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "id": 18
};

fetch(url, {
    method: "DELETE",
    headers,

UPDATE Order the pictures by sort.

Example request:
curl --request PATCH \
    "http://127.0.0.1:8000/api/sort" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"data\": [
        \"quidem\"
    ],
    \"data[0][\'id\']\": 20,
    \"data[0][\'sort\']\": 6
}"

const url = new URL(
    "http://127.0.0.1:8000/api/sort"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "data": [
        "quidem"
    ],
    "data[0]['id']": 20,
    "data[0]['sort']": 6
};

fetch(url, {
    method: "PATCH",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
